# 🌐 Personal Portfolio Website

A modern and fully responsive **React-based portfolio website** built to showcase my projects, skills, and experience.  
This portfolio includes smooth animations, interactive UI elements, and a working contact form.

---

## 🚀 Tech Stack

- ⚛️ React.js
- 🎨 Tailwind CSS
- 🎥 Framer Motion
- 📧 EmailJS
- 🌐 React Router DOM

---

## ✨ Features

- 🔥 Smooth intro animation on page load
- 🎬 Advanced scroll-based animations using Framer Motion
- 📱 Fully responsive design (Mobile + Tablet + Desktop)
- 📧 Functional Contact Form using EmailJS
- 🎯 Clean UI with modern layout
- ⚡ Fast performance and optimized components
- 🧩 Component-based architecture

---

## 🎞 Animations Used

- Page transition animations
- Fade-in & slide-in effects
- Scroll-triggered animations
- Hover effects on buttons and cards
- Intro loading animation

📧 EmailJS Configuration

To make the contact form work:
Create an account on EmailJS.
Create a new email service.
Create an email template.
Add your:
Service ID
Template ID
Public Key

Store them inside a .env file:
VITE_SERVICE_ID=
VITE_TEMPLATE_ID=
VITE_PUBLIC_KEY=

## 🛠 Installation & Setup

Clone the repository:

git clone https://github.com/your-username/your-portfolio.git
Navigate to the project folder:
cd your-portfolio
Install dependencies:
npm install
Run the development server:
npm run dev
